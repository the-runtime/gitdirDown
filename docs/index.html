GitdirDown - Support

About GitdirDown

GitdirDown is a powerful web extension that enables users to download folders from GitHub repositories with just a few clicks. With GitdirDown, you can effortlessly retrieve specific directories from GitHub and save them directly to your computer.

How to Use GitdirDown

  1. Install the GitdirDown web extension from the Firefox Add-ons store.
  2. Navigate to a GitHub repository and find the folder you want to download.
  3. Right-click on the folder's link and choose "GitdirDown - Download GitHub Folder" from the context menu.
  4. The download will start automatically, and the folder will be saved to your computer.

Frequently Asked Questions

1. Can I download multiple folders at once?

No, GitdirDown currently supports downloading one folder at a time. You can repeat the process for each folder you want to download.

2. Where are the downloaded folders saved?

The downloaded folders are typically saved in your default download directory. You can check your browser's settings to customize the download location.

3. How can I report a bug or suggest a feature?

We appreciate your feedback! Please visit our GitHub repository and create an issue with a detailed description of the problem or feature request.
